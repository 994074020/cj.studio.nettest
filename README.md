
# nettest

## 网络测试中心

- 支持的协议有：ws,tcp,udt。http协议（restful)的测试
- 测试数据和报告存入平台的测试库
- 适合大企业建立测试中心
- 每个人管理自己的测试数据
- 所有服务方法全中心可见
